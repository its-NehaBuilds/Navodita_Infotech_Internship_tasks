# 📌 1. Data Structures & Algorithms Visualizer

## 📊 Overview
A console-based C++ application that visualizes sorting algorithms step-by-step using arrays and bar representations in the terminal.

It helps learners understand how sorting algorithms work internally by showing comparisons, swaps, and array transformations.

---

## ✨ Features
- Bubble Sort visualization  
- Selection Sort visualization  
- Insertion Sort visualization  
- Merge Sort implementation  
- Quick Sort implementation  
- Random array generator  
- Step-by-step execution display  
- Bar-based visualization in console  
- Menu-driven interface  

---

## 🛠 Tech Stack
- C++
- Arrays
- OOP concepts (basic class structure)
- Console-based UI

---

## ⚙️ How It Works
1. Generate random array
2. Display menu
3. Select sorting algorithm
4. Algorithm runs step-by-step
5. Visual representation updates after each operation
6. Option to generate new array

---

## 📁 Files
- `main.cpp` → Program control & menu
- `sorting.cpp` → Sorting algorithms
- `sorting.h` → Function declarations
- `utils.cpp` → Array generation & visualization
- `utils.h` → Utility functions
